# Aponic Discord Bot Privacy Policy

Last updated: June 25, 2026

## 1. Introduction

This Privacy Policy explains what data Aponic may collect, store, and use. Aponic is the name used for the official Aponic Discord Bot.

This Policy applies only to the official Aponic Discord Bot and the features it provides inside Discord.

## 2. Data We May Collect

Depending on which features are enabled, Aponic may collect or store:

- Discord server IDs
- Discord channel IDs
- Discord user IDs when needed for bot features
- Discord role IDs when needed for permissions or role features
- Roblox group IDs
- Roblox user IDs
- Roblox usernames
- buyer names
- buyer IDs
- item IDs
- item names
- Robux amounts
- sale timestamps
- sales reports
- bot configuration values

## 3. Discord Data

Aponic may use Discord server IDs, channel IDs, user IDs, and role IDs to operate bot features in Discord.

This data may be used for permissions, role features, command behavior, logging, reports, analytics, member counts, and bot configuration.

## 4. Roblox Data

Aponic may use Roblox group IDs, Roblox user IDs, Roblox usernames, Robux counts, sale information, item information, and related Roblox data to operate enabled Roblox-related features.

Some Roblox-related features may require authentication credentials or cookies to access Roblox group data. These sensitive values are used only to operate the bot features that require them.

## 5. Sales and Analytics Data

Aponic may collect or store buyer names, buyer IDs, item IDs, item names, Robux amounts, sale timestamps, sales reports, and analytics values when sales logs, reports, or analytics features are enabled.

This data is used only to operate bot features such as sales logs, reports, analytics, member counts, Robux counts, role features, and bot configuration.

## 6. How We Use Data

Aponic uses collected data only to operate, configure, and maintain the bot features that are enabled for a Discord server.

Examples include sending logs to configured channels, generating reports, checking permissions, managing role features, tracking counts, and saving bot configuration values.

## 7. Data Storage

Aponic data may be stored in local files, SQLite databases, server storage, or the hosting environment used to run the bot.

Sensitive secrets such as Discord tokens, Roblox cookies, and API tokens are configuration secrets and should never be shared publicly.

## 8. Data Sharing

Aponic does not sell stored bot data and does not share stored bot data with advertisers.

Data may be processed through Discord, Roblox, hosting providers, or other services required to run the bot and provide enabled features.

## 9. What We Do Not Sell

Aponic does not sell Discord IDs, Roblox IDs, Roblox usernames, sales data, analytics data, bot configuration values, or other stored bot data.

## 10. Security

Reasonable steps should be taken to protect stored bot data and sensitive configuration secrets.

No system can be guaranteed to be completely secure. Server owners and administrators should keep tokens, cookies, credentials, and configuration secrets private.

## 11. Data Deletion Requests

Users can request deletion of stored bot data through the contact method listed below.

Deletion requests may require enough information to identify the relevant Discord server, Discord user, Roblox group, or stored bot configuration.

## 12. Third-Party Services

Aponic is not affiliated with Discord or Roblox.

Some features depend on Discord and Roblox services. Those services may collect, store, or process data under their own terms and privacy policies.

## 13. Changes to this Policy

This Privacy Policy may be updated from time to time. Updates will be posted in this repository with a new last updated date.

Your continued use of Aponic after changes are posted means you accept the updated Privacy Policy.
